  1. 技能倍率：指角色天赋中详细属性栏中的数据

  2. 角色自身攻击力：角色本身也有攻击力数值

  3. 武器攻击力：武器主词条攻击力

  4. 攻击力百分比加成：

    1. 指武器攻击力百分比副词条

    2. 圣遗物主副词条（攻击力百分比）

    3. 圣遗物效果：角斗士两件套、追忆两件套、如昔日宗室4件套

    4. 讨龙英杰谭特效

    5. 双火元素共鸣等等

  5. 攻击力数值加成：圣遗物羽毛、角色技能（如班尼特、九条裟罗）

  6. 激化反应增加：在触发超激化、蔓激化时，基础伤害会有激化反应的增加值，否则激化反应增加值为0，详细参见激化反应。

常说的白字攻击力（基础攻击力）指的就是：角色自身攻击力 + 武器攻击力

而绿字攻击力就是：(角色自身攻击力+武器攻击力) × 攻击力百分比加成 + 攻击力数值加成

元素反应

元素类别

  1. 水火草冰雷风岩
  2. 燃元素：详见燃烧反应
  3. 冻元素：详见冻结反应

元素附着

  • 计时器，2.5秒冷却

  • 计数器，2.5秒内每3次才能一次附着

  • 计时计数器在怪物上

  • 不同技能计时计数器不同

  • 元素附着量

    • 强，2单位，留存时间12s
      • 班尼特EQ、北斗E、万叶长E和Q、温迪E、琴E、
    • 弱，1单位，留存时间9.5s
    • 不论强弱，附着到怪物身上时还要再乘0.8，如果附着前有其他元素可以相互消耗则按1单位算

元素克制

表示不同元素之间可以相互消耗，并且消耗比例不同，否则可以共存。元素反应与元素间是否消耗没有关联。

  • 水克火，一单位水消耗两单位火
  • 火克冰，一单位火消耗两单位冰
  • 草克水，一单位草消耗两单位水

由此可以得出：破火盾水元素最快、破冰盾火元素最快、破水盾草元素最快

后手元素不附着

如果附着的元素参与了反应，但是该元素没有被消耗完，那么他也不会留存在怪物身上

元素反应

增幅反应(蒸发融化)

指蒸发与融化，具体查看增幅反应区

剧变反应

剧变反应指额外造成的伤害，伤害数值只与反应系数和引发反应角色的等级、精通、怪物抗性有关。

同种剧变反应对同种怪物每0.5秒最多2次伤害，超载1次。

冻结、原激化、结晶不会造成伤害。

冻结

水+冰 = 冻元素

产生效果：使怪物冻结

冻元素有以下特点：

  1. 由水冰元素1：1：1产生，表现效果为冻结，图标如下

  2. 与冰水不反应，因此会有冻冰共存，冻水共存。但是图标都是冻元素图标，因为看不出来所以会见到藏冰，藏水的说法。

  3. 与风火雷岩反应，表现与冰元素类似

  4. 与风反应时，如果冻与冰或水共存，则会优先扩冰或扩水

  5. 碎冰反应：处于冻元素影响下的敌人如果受到钝击（大剑攻击，重击， 可莉普攻重击）或岩属性的攻击时，退出冻结状态（冻元素消失），并产生额外的物理伤害

激化反应

激化反应的机制就是，先使用草类元素触发原激化使怪物进入激化状态，怪物在激化状态时，有以下两种效果：

  • 受到雷元素伤害时：触发超激化，基于触发者等级、精通和相关超激化反应加成，增加基础伤害（是一个整体的加操作）
  • 受到草元素伤害时：触发蔓激化，基于触发者等级、精通和相关蔓激化反应加成，增加基础伤害（是一个整体的加操作）
原激化

草+雷 = 激元素

由草雷元素1：1：1产生，表现效果为黄色闪电，图标与草元素相同，不会造成伤害，但会产生激元素，使怪物进入激化状态。

激元素有以下特点：

  1. 激元素附着量随时间衰减、弱激元素一般持续10s左右
  2. 与雷产生超激化反应，但不会消耗激元素，可激雷共存
  3. 与草产生蔓激化反应，但不会消耗激元素，可激草共存（但由于图标相同所以不会直接看出来，证据为激草共存下的怪物受到雷元素伤害时产生了原激化与超激化）
  4. 与火燃烧，与水绽放，表现与草类似
  5. 与冰风岩不反应
超激化、蔓激化

超激化：激元素 + 雷

蔓激化：激元素 + 草

系数：超激化1.15、蔓激化1.25

产生效果：增加基础伤害数值，但不会消耗激元素（激元素会自身的恒定的速率衰减）

伤害影响参数：触发者角色等级、精通、装备特效、是否暴击、怪物防御力、怪物雷元素抗性（超激化时）或草元素抗性（蔓激化时）
